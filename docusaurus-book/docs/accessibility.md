# Accessibility Statement for Physical AI & Humanoid Robotics Book

We are committed to making this book accessible to all users, including those with disabilities. This book is designed to meet WCAG 2.1 AA standards.

## Accessibility Features

- **Semantic HTML**: Proper heading hierarchy (H1, H2, H3) for screen readers
- **Alt text**: All images include descriptive alt text
- **Keyboard navigation**: All interactive elements are accessible via keyboard
- **Color contrast**: All text meets WCAG 2.1 AA contrast requirements
- **Focus indicators**: Clear focus indicators for keyboard navigation
- **Screen reader support**: Proper ARIA labels and roles where needed

## Content Accessibility

- All code examples include accessible descriptions
- Diagrams and illustrations have detailed alt text
- Interactive components provide text alternatives
- Math equations use proper markup for screen readers

## Feedback

If you encounter accessibility issues or have suggestions for improvement, please contact us through our GitHub repository.
